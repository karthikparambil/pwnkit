# auto-num.py

`auto-num.py` is a simple Python script that generates a list of numbers within a user-defined range and saves them to a specified file. It is useful for creating custom number wordlists or generating sequential number lists for pentesting and cybersecurity tasks.

## Features

- Generates numbers from a specified start to end range.
````Start: 1
end: 100
Enter filename: 100.txt
File saved as: 100.txt
````         
- Writes the generated numbers to a user-specified file.
- Simple and easy to use for creating custom number sequences.

## Requirements

- **Python 3.x**: This script requires Python to be installed on your system.

## Usage

1. Download the script.
2. Run the script:

```
   python3 auto-num.py
```
